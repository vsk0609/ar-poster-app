# AR Poster App

An Augmented Reality (AR) web application that displays 3D content when a marker is detected using your camera.

## Features
- Marker-based AR using AR.js
- 3D rotating object (A-Frame)
- Real-time camera interaction
- Clean UI overlay

## How to Use
1. Open the app in a browser
2. Allow camera access
3. Show a Hiro marker to the camera
4. Watch the 3D object appear and rotate

## Tech Stack
- HTML
- CSS
- JavaScript
- A-Frame
- AR.js

## Future Improvements
- Custom markers
- 3D models (GLTF)
